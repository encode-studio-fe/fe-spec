# encode-fe-eslint-plugin

# 安装

除了本包，你需要同时安装 ESlint (opens new window)

$ npm install encode-fe-eslint-plugin eslint --save-dev

# 使用

# 引入插件

// .eslintrc.js
module.exports = {
  plugin: ['encode-fe-eslint-config'],
  rules: {
    'encode-fe-eslint-plugin/no-secret-info': 'error',
  },
};

# 使用 presets

// .eslintrc.js
module.exports = {
  extends: 'plugin:encode-fe-eslint-plugin/recommended',
};

# 支持的规则

  • no-broad-semantic-versioning (opens new window) 不要指定宽泛的版本范围
  • no-http-url (opens new window) 使用 HTTPS 协议头的 URL，而不是 HTTP
  • no-js-in-ts-project (opens new window) 不要在 TS 项目中使用 JS
  • no-secret-info (opens new window) 不要在代码中直接设置 password token and secret 信息
